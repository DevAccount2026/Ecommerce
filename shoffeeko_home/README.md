# ShoffeeKo Modular Home Page

Open `index.html` using VS Code Live Server. The page only assembles modules; it does not merge everything into one file.

## Structure
- `index.html` = home page assembler
- `assets/css/base.css` = global styles/header/footer
- `assets/js/module-loader.js` = loads component HTML/CSS/JS/JSON
- `components/<module>/` = isolated module files
- `assets/images/` = temporary screenshot placeholders
- `assets/videos/` = add MP4 files here later

Replace placeholder images/video paths inside each module JSON when your final assets are ready.
